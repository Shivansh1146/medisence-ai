# MedicSense AI

Intelligent Healthcare Triage & Appointment Assistant.

## Overview
MedicSense AI is an AI-powered healthcare assistant that provides symptom triage using Google Gemini API and manages appointments.

## Architecture
- **Frontend**: Next.js
- **Backend**: FastAPI
- **AI**: Google Gemini
- **Database**: PostgreSQL (Cloud SQL)
- **Infrastructure**: Google Cloud Platform (Cloud Run)
